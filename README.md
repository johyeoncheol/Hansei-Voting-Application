# Hansei-Voting-Application

간단한 투표 앱입니다. 투표, 건의함, 게시판 등등의 기능을 제공합니다.

## 기능

1. 이메일 로그인 (파이어베이스 이용)
2. 투표 생성 (제목, 투표 항목, 시간 설정등등)
3. 투표 리스트
4. 공지사항
5. 건의함
6. 설정화면
7. 캘린더 기능(미완성)
